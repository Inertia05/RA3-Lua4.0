# Corona Mod LUA 4.0 Scripts for Custom Maps

This repository contains LUA 4.0 scripts and tools specifically designed for use in unofficial, custom maps developed for the *Corona* mod, a popular *Red Alert 3* modification celebrated for its Chinese-inspired themes.

Interestingly, music from the *Corona* mod gained widespread recognition when it was featured during the 2024 Olympic Games and a live broadcast from China’s Tiangong Space Station on CCTV. This highlights the mod's cultural impact beyond the game, with its music resonating with a global audience during major real-world events.
